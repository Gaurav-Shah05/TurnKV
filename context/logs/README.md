# Worklogs

One file per person per week. Informal — what you worked on, what's blocking you, what you want someone else's eyes on.

Filename convention: `YYYY-WW-<firstname>.md` (ISO week number).

Example: `2026-16-gaurav.md` for the week of April 13–19, 2026.

Link PRs, ADRs (`decisions/…`), and experiments (`experiments/…`) by relative path so others can jump to what you're talking about.
